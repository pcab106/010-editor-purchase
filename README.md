## 010 Editor 是什么，为什么值得买正版

010 Editor 是加拿大 SweetScape Software 出品的专业文本与十六进制编辑器，自 2003 年发布以来一直是这个细分领域的标杆工具。

普通的十六进制编辑器能干什么？打开文件，看字节，改字节，完事。

010 Editor 能干什么？它有一套叫"二进制模板"（Binary Template）的系统——你可以用类 C 语法写模板，把一个二进制文件解析成有层次的数据结构，就像给它做了一个「X 光透视」。想看一个 BMP 图片的文件头怎么组织？想分析一个 ZIP 的内部结构？打开模板，一目了然，还能直接在结构化视图里改值。

这个功能在逆向工程、固件分析、数字取证领域简直是神器级别的存在。官方仓库里有社区贡献的几百个现成模板，常见格式基本都覆盖了。

除了模板，它还有：

- 支持编辑无限大小的文件（TB 级别不是问题）
- 文件逐字节差异对比
- 直方图数据可视化
- 支持编辑磁盘驱动器（NTFS、FAT32 等）
- 脚本自动化（写个脚本批量处理二进制文件，非常爽）
- 支持 Windows / macOS / Linux 全平台
- 文本编辑同样能打：XML、HTML、C/C++、PHP、UTF-8 等多种格式语法高亮
